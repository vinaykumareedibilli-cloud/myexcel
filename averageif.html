AVERAGEIF: Conditional Average (Single Criterion)

The **AVERAGEIF** function calculates the **average (arithmetic mean)** of all cells in a range that meet a given single **criterion**. It combines the power of AVERAGE with conditional logic.

**Key Use:** Finding the average sales per day of the week, or the average score for students who passed (score > 60).

---

Formula Syntax

=AVERAGEIF(range, criteria, [average_range])
---

Arguments (Parts of the Formula)

  • range: The range of cells that you want to apply the **criteria** against. (e.g., Column A, which lists regions).
  • criteria: The condition that defines which cells in the `range` will be averaged. (e.g., `"North"`, `"<100"`).
  • [average_range]: *Optional*. The actual range of cells that contain the values to average. If omitted, the `range` is used as the average range. (e.g., Column B, which lists sales figures).
---

Example üí°

To find the average unit price (Column C) for all products listed as **"Small"** (Column B):

=AVERAGEIF(B:B, "Small", C:C)
