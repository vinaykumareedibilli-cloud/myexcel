CONCAT: Joining Text Strings

The **CONCAT** function (introduced in newer Excel versions, replacing CONCATENATE) joins two or more text strings into one string. It's the standard way to merge content from different cells or add custom text.

**Key Use:** Combining first and last names, creating a full address from separate cells, or generating custom report titles.

---

Formula Syntax

=CONCAT(text1, [text2], ...)

**Alternative:** You can also use the **ampersand (&)** symbol to join text: =text1 & text2

---

Arguments (Parts of the Formula)

  • text1: The first text item to join. This can be a cell reference (e.g., `A2`) or a text string enclosed in quotes (e.g., `"The total is: "`).
  • [text2], ...: Additional text items to join. You can include up to 255 items.
---

Example üí° (Joining Names)

To combine the first name in A2, a space, and the last name in B2:

=CONCAT(A2, " ", B2)

Example üí° (Using Ampersand)

The equivalent using the ampersand (&) operator:

=A2 & " " & B2
