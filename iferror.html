IFERROR: Handling Errors Gracefully

The **IFERROR** function simplifies error trapping. It checks the value of a formula; if the formula evaluates to an error (like `#N/A`, `#DIV/0!`, `#VALUE!`), it returns a specified result. Otherwise, it returns the result of the formula itself.

**Key Use:** Preventing messy error values when using LOOKUP formulas or division, or displaying a custom message like "Data Missing."

---

Formula Syntax

=IFERROR(value, value_if_error)
---

Arguments (Parts of the Formula)

  • value: The formula, cell reference, or calculation you want to check for an error.
  • value_if_error: The value to return if the `value` argument results in any error. This is usually text (e.g., `"Not Found"`) or zero (`0`).
