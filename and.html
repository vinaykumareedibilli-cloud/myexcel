AND: Checking Multiple Conditions (All Must be TRUE)

The **AND** function checks if **all** of the arguments (conditions) are true. It returns **TRUE** only if *every single condition* is met. If even one condition is false, it returns **FALSE**.

**Key Use:** AND is typically nested inside an **IF** function to test complex rules (e.g., "If the profit is > 10% **AND** the inventory is < 50, then reorder.").

---

Formula Syntax

=AND(logical1, [logical2], ...)
---

Arguments (Parts of the Formula)

  • logical1: The first condition you want to test that can be evaluated to TRUE or FALSE. (e.g., `A2 > 100`).
  • [logical2], ...: *Optional*. Additional conditions you want to test. Up to 255 conditions can be included. (e.g., `B2 < 5000`).
---

Example üí°

To return TRUE if the score (C5) is above 70 **AND** the status (D5) is "Complete." If either condition fails, it returns FALSE.

=AND(C5 > 70, D5 = "Complete")

**Example nested in IF:** =IF(AND(C5 > 70, D5 = "Complete"), "Passed All Rules", "Failed Rule")
