IF: Checking a Single Condition (The Core Logical Function)

The **IF** function is the most fundamental logical function in Excel. It checks whether a condition is met, and returns one value if the condition is **TRUE** and another value if it is **FALSE**.

**Key Use:** Categorizing data (Pass/Fail, High/Low), applying different calculations based on a rule, or setting up conditional formatting.

---

Formula Syntax

=IF(logical_test, value_if_true, [value_if_false])
---

Arguments (Parts of the Formula)

  • logical_test: Any value or expression that can be evaluated to TRUE or FALSE. This is the **condition** you are testing. (e.g., `C2 >= 70`).
