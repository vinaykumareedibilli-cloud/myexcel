SORT & SORTBY: Dynamic Sorting

The **SORT** and **SORTBY** functions are **Dynamic Array Formulas** that return a sorted version of a range of data. Unlike manual sorting, these functions create a new, live-updating output.

**Key Difference:** **SORT** sorts by the columns *within the array itself*. **SORTBY** sorts an array based on the values in a *separate* column or range.

---

1. SORT Function (Sorting by Columns Within the Array)

The simplest way to sort a range of data by one of its columns.

=SORT(array, [sort_index], [sort_order], [by_col])
  • array: The range to sort (e.g., the entire table `A2:D100`).
  • [sort_index]: *Optional*. The column number in the `array` to sort by (1 for the first column, 2 for the second, etc.). Default is 1.
  • [sort_order]: *Optional*. **1** for Ascending (default) or **-1** for Descending.

Example üí° (SORT)

To sort the table A2:D100 by the second column (Column B) in descending order:

=SORT(A2:D100, 2, -1)
---

2. SORTBY Function (Sorting by External/Custom Columns)

Sorts an array using one or more columns that may not be contained within the array itself.

=SORTBY(array, by_array1, [sort_order1], [by_array2, sort_order2], ...)
  • array: The range you want to return (e.g., `A2:A100`).
  • by_array1: The range containing the values you want to sort by (e.g., the Sales column, `D2:D100`).
  • [sort_order1]: **1** (Ascending) or **-1** (Descending) for the first sort array.

Example üí° (SORTBY)

To sort just the names in Column A based on their corresponding Sales values in Column D (descending):

=SORTBY(A2:A100, D2:D100, -1)

**Combined Use:** You can nest **FILTER** and **SORT** to dynamically extract and order data in a single step!
