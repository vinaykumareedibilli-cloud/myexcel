TODAY & NOW: Dynamic Date and Time

The **TODAY** and **NOW** functions are non-argument functions that return the current system date and/or time. They are **volatile**, meaning they recalculate every time the workbook is opened or a change is made, ensuring the displayed date/time is always current.

**Key Use:** Calculating the number of days until a deadline, determining the age of an invoice, or providing an automated timestamp for reports.

---

1. TODAY Function

Returns the **current date** only, formatted as a date. The result is equivalent to the **Integer** part of the internal Excel date value (serial number).

=TODAY()
  • Arguments: None. You must include the empty parentheses.

Example üí° (Calculating Age in Days)

If a person's birthdate is in cell A2, you can calculate their current age in days:

=TODAY() - A2
---

2. NOW Function

Returns the **current date and time**, formatted as a date and time value. The result is the **Full** internal Excel date value (serial number, including the decimal fraction for time).

=NOW()
  • Arguments: None. You must include the empty parentheses.

Example üí° (Timestamping a Report)

To display a timestamp showing when the workbook was last recalculated:

="Report Last Updated: " & TEXT(NOW(), "yyyy-mm-dd hh:mm AM/PM")
