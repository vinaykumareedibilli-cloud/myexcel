DCOUNTA: Conditional Count of Non-Blank Cells

The **DCOUNTA** (Database Count All) function counts the cells that are **not blank** within a specified column (field) of a database or list, but only for the rows that meet a defined set of **criteria**.

**Key Use:** Counting records in a database that satisfy multiple complex conditions. Unlike `COUNTIF/COUNTIFS`, the D functions use a separate criteria range, making complex conditions much easier to manage.

---

Formula Syntax

=DCOUNTA(database, field, criteria)

**DCOUNT vs. DCOUNTA:** **DCOUNT** counts only **numerical** cells. **DCOUNTA** counts **all non-blank** cells, including numbers, text, and dates.

---

Arguments (Parts of the Formula)

  • database: The range that makes up the list or database, including the **header row** (e.g., `A1:E100`).
  • field: The column to count. This can be the **column header name** (as text in quotes, e.g., "Customer ID") or the **number** corresponding to the column's position in the database (e.g., 3 for the third column).
  • criteria: The range that contains the criteria, which must include **at least one column label** (header) and **at least one cell below it** specifying the condition (e.g., `G1:G2`).
---

Example üí°

You have a sales database in A1:D100. Column C is "Region" and Column D is "Sales Rep" (which contains some blank cells). You want to count all records where the **Region is "West"** (Criteria set up in G1:G2) that have an entry in the "Sales Rep" column.

=DCOUNTA(A1:D100, "Sales Rep", G1:G2)

**Criteria Setup:** The range **G1:G2** would need to have the column header **"Region"** in G1 and the value **"West"** in G2.
