ISERROR & ISBLANK: Logical Checks

The **IS** functions check the type of a value or cell contents and return a simple **TRUE** or **FALSE**. They are most commonly nested inside an **IF** function to trigger an action based on whether an error or blank cell is found.

---

1. ISERROR Function

Checks if a value is **any type of error** (`#N/A`, `#VALUE!`, `#REF!`, `#DIV/0!`, `#NUM!`, `#NAME?`, or `#NULL!`) and returns **TRUE** if it is.

=ISERROR(value)
  • value: The value or formula you want to check for an error. This is typically a cell reference (e.g., `A1`) or another complex function.

Example üí° (Error Trapping)

Use `ISERROR` inside an `IF` statement to display a dash instead of the `#DIV/0!` error:

=IF(ISERROR(A2/B2), "-", A2/B2)

**Modern Alternative:** The function **IFERROR** is a simpler way to achieve the same result: `IFERROR(A2/B2, "-")`.

---

2. ISBLANK Function

Checks if a cell is **completely empty** (contains no data, no formula, and no invisible character) and returns **TRUE** if it is.

=ISBLANK(value)
  • value: The cell reference you want to check (e.g., `C5`).

Example üí° (Checking for Missing Data)

Only calculate a commission if the Sales Amount (C5) is not blank:

=IF(ISBLANK(C5), "Missing Data", C5 * 0.1)
