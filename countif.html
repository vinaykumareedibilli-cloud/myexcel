COUNTIF: Count with Single Condition

The **COUNTIF** function counts the number of cells within a specified **range** that satisfy a single **criterion**. It's used to quickly get a tally or frequency count of items.

**Key Use:** Counting how many employees are in the "Marketing" department, or counting how many invoices are "Overdue."

---

Formula Syntax

=COUNTIF(range, criteria)
---

Arguments (Parts of the Formula)

  • range: The range of cells that you want to count. This is the column or row that contains the item you are looking for. (e.g., Column B, which lists all statuses).
  • criteria: The condition that defines which cells in the `range` will be counted. This can be text, a number, or a comparison operator like `">50"`. (e.g., `"Complete"`).
---

Example üí°

To count how many projects listed in Column A have a status of "Complete":

=COUNTIF(A:A, "Complete")
