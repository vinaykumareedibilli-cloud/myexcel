GETPIVOTDATA: Extracting Data from PivotTables

The **GETPIVOTDATA** function retrieves summary data stored within a **PivotTable**. It is highly reliable because it references the data by its **name** (e.g., "South Region Sales") rather than its cell coordinate, so the formula won't break if the PivotTable layout changes.

**Key Use:** Building reports or dashboards that pull specific, stable metrics from a volatile PivotTable source.

---

Formula Syntax

=GETPIVOTDATA(data_field, pivot_table, [field1, item1], [field2, item2], ...)

**Pro Tip:** The easiest way to create this formula is to type `=` in any cell, then click the cell in the PivotTable that contains the data you want. Excel will generate the formula automatically.

---

Arguments (Parts of the Formula)

  • data_field: The name of the value field you want to retrieve. This must be the **exact name** shown in the PivotTable (e.g., "Sum of Sales" or "Count of Orders"). Must be enclosed in quotes.
  • pivot_table: A reference to **any cell** within the PivotTable (e.g., `A4`).
  • [field1, item1], ...: *Optional, but typically required*. Pairs of arguments that describe the specific filter or data point you want.
    • **Field:** The name of the PivotTable row, column, or filter label (e.g., "Region").
    • **Item:** The specific value within that field you want to filter by (e.g., "East").
---

Example üí°

Retrieve the "Sum of Sales" from a PivotTable (starting in A3) for the "North" region and the "Q4" quarter:

=GETPIVOTDATA("Sum of Sales", A3, "Region", "North", "Quarter", "Q4")

**Dynamic Example:** If you have the Region name in cell **B1** and the Quarter in cell **C1**, you can make the formula dynamic:

=GETPIVOTDATA("Sum of Sales", A3, "Region", B1, "Quarter", C1)
