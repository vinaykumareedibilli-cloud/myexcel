SEQUENCE: Generating a List of Numbers

The **SEQUENCE** function is a **Dynamic Array Formula** that generates a list of sequential numbers into an array (a range of cells). This list can be a single column, a single row, or a rectangular grid.

**Key Use:** Quickly creating numbering for reports, generating a list of dates, or setting up complex dynamic data grids.

---

Formula Syntax

=SEQUENCE(rows, [columns], [start], [step])

**Dynamic Array Note:** Enter the formula once in the top-left cell; the results "spill" automatically into adjacent cells.

---

Arguments (Parts of the Formula)

  • rows: The number of rows of data to return.
  • [columns]: *Optional*. The number of columns of data to return. Default is 1.
  • [start]: *Optional*. The starting number of the sequence. Default is 1.
  • [step]: *Optional*. The amount to increment each subsequent value in the sequence. Default is 1.
---

Example üí° (Simple Column List)

To generate a numbered list of 10 items in a single column, starting at 1:

=SEQUENCE(10)

Example üí° (Dynamic Grid)

To create a 5-row by 3-column grid, starting at 100 and counting up by 10 (100, 110, 120, etc., row-by-row):

=SEQUENCE(5, 3, 100, 10)

Example üí° (Generating Dates)

To generate the next 30 days starting from today's date (where TODAY() returns a number treated as a date):

=TODAY() + SEQUENCE(30)
