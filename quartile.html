LARGE & SMALL: Finding the Nth Largest/Smallest Value

The **LARGE** and **SMALL** functions retrieve a value from a dataset based on its relative rank. They are commonly used to find top performers or bottom outliers.

**Key Difference:** **MAX** and **MIN** find *only* the 1st largest/smallest value. **LARGE** and **SMALL** can find the 1st, 2nd, 3rd, 10th, etc., largest or smallest value.

---

1. LARGE Function (Nth Largest)

Returns the **k-th largest value** in a data set. For example, the 3rd largest score.

=LARGE(array, k)
  • array: The range of numerical data from which to find the largest value (e.g., `B2:B100`).
  • k: The position (from the largest) in the array to return. Use `1` for the maximum, `2` for the second largest, and so on.

Example üí° (LARGE)

To find the **second largest** sales figure from the range C2:C100:

=LARGE(C2:C100, 2)
---

2. SMALL Function (Nth Smallest)

Returns the **k-th smallest value** in a data set. For example, the 4th slowest time.

=SMALL(array, k)
  • array: The range of numerical data from which to find the smallest value.
  • k: The position (from the smallest) in the array to return. Use `1` for the minimum, `2` for the second smallest, and so on.

Example üí° (SMALL)

To find the **third smallest** price from the range D2:D50:

=SMALL(D2:D50, 3)
