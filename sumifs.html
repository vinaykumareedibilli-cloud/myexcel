SUMIFS: Conditional Summation (Multiple Criteria)

The **SUMIFS** function adds the values in a range that meet **multiple** specified criteria. This is the function you use when you need to sum data based on two or more rules simultaneously.

**Key Difference from SUMIF:** In SUMIFS, the **sum_range** argument comes *first*.

---

Formula Syntax

=SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...)
---

Arguments (Parts of the Formula)

  • sum_range: The actual cells to sum (e.g., the column containing prices or quantities). This is the **required first argument**.
  • criteria_range1: The first range to apply a test against (e.g., Column A, listing Regions).
  • criteria1: The first condition (e.g., `"North"`).
  • [criteria_range2, criteria2], ...: *Optional*. Subsequent pairs of ranges and their corresponding criteria. You can add up to 127 pairs.
---

Example üí°

Find the total sales revenue (Column D) for the "North" region (Column A) **AND** for the product "Laptop" (Column B):

=SUMIFS(D:D, A:A, "North", B:B, "Laptop")
