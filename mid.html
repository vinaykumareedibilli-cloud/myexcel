MID: Extracting Text from the Middle

The **MID** function returns a specific number of characters from a text string, starting at the position you specify. It is the most flexible of the extraction functions (LEFT, RIGHT, MID).

**Key Use:** Isolating variable-length product numbers, extracting month codes from dates embedded in text, or separating text between two delimiters.

---

Formula Syntax

=MID(text, start_num, num_chars)
---

Arguments (Parts of the Formula)

  • text: The text string or cell reference containing the text you want to manipulate (e.g., `A2`).
  • start_num: The position of the first character you want to extract. The first character of the text is position 1.
  • num_chars: The number of characters you want the MID function to return.
---

Example üí°

If A2 contains **"LND-Q4-2025-001"**, this extracts the quarter code ("Q4"):

=MID(A2, 5, 2) → Result: **"Q4"**

**Explanation:** The function starts at the 5th character (`Q`) and extracts 2 characters (`Q` and `4`).
