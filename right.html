LEFT & RIGHT: Extracting Text from Ends

The **LEFT** and **RIGHT** functions allow you to extract a specific number of characters from the beginning (left side) or the end (right side) of a text string.

**Key Use:** Isolating codes, extracting initials, or separating fixed-length parts of an identifier (e.g., getting the year from a part number).

---

1. LEFT Function

Retrieves characters starting from the **left** side of a text string.

=LEFT(text, [num_chars])
  • text: The text string or cell reference containing the text you want to manipulate (e.g., `A2`).
  • [num_chars]: *Optional*. The number of characters you want to extract, counting from the left. Default is 1.

Example üí°

If A2 contains **"XYZ-456-ABC"**, this extracts the first three characters:

=LEFT(A2, 3) → Result: **"XYZ"**
---

2. RIGHT Function

Retrieves characters starting from the **right** side of a text string.

=RIGHT(text, [num_chars])
  • text: The text string or cell reference containing the text you want to manipulate (e.g., `A2`).
  • [num_chars]: *Optional*. The number of characters you want to extract, counting from the right. Default is 1.

Example üí°

If A2 contains **"XYZ-456-ABC"**, this extracts the last three characters:

=RIGHT(A2, 3) → Result: **"ABC"**
