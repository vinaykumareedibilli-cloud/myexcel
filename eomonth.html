EOMONTH: Calculating the End of the Month

The **EOMONTH** function is used to calculate the date of the **last day of the month** that is a specified number of months in the future or past from a given `start_date`.

**Key Use:** Setting due dates (e.g., invoices always due on the last day of the month), calculating the end date of fiscal quarters, or determining subscription renewal dates.

---

Formula Syntax

=EOMONTH(start_date, months)

**Result Format:** EOMONTH returns a date as a **serial number**. Be sure to format the cell as a **Date** for a readable result.

---

Arguments (Parts of the Formula)

  • start_date: The initial date. Can be a cell reference (e.g., `A2`) or a date function (e.g., `TODAY()`).
  • months: A positive or negative whole number representing the number of months before or after the `start_date`.
    • **0:** Returns the end of the same month as the `start_date`.
    • **1:** Returns the end of the next month.
    • **-1:** Returns the end of the previous month.
---

Example üí° (End of Current Month)

To find the last day of the current month, regardless of the current day:

=EOMONTH(TODAY(), 0)

Example üí° (Quarterly Due Date)

If the invoice date is in **A2**, find the due date, which is the end of the month **three months later**:

=EOMONTH(A2, 3)
