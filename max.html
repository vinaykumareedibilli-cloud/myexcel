MAX: Finding the Largest Number

The **MAX** function finds and returns the **largest numerical value** in a specified set of values or a range of cells. This is essential for quickly identifying peak sales, highest scores, or maximum temperatures.

**Note:** MAX only considers numbers. It ignores empty cells, logical values (TRUE/FALSE), and text.

---

Formula Syntax

=MAX(number1, [number2], ...)
