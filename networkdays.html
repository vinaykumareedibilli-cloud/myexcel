NETWORKDAYS: Calculating Working Days

The **NETWORKDAYS** function calculates the number of whole **working days** between two dates. By default, it excludes **Saturdays and Sundays** and can optionally exclude a list of holidays.

**Key Use:** Project scheduling, calculating delivery lead times, and measuring employee time cards.

---

Formula Syntax

=NETWORKDAYS(start_date, end_date, [holidays])

**Note:** If you need to specify different weekend days (e.g., Friday/Saturday), you must use the **NETWORKDAYS.INTL** function instead.

---

Arguments (Parts of the Formula)

  • start_date: The initial date of the period.
  • end_date: The final date of the period.
  • [holidays]: *Optional*. A range of cells containing dates that should also be excluded from the count (e.g., statutory holidays).
---

Example üí° (Simple Working Days Count)

To find the number of working days between January 1, 2025 (in A2) and January 31, 2025 (in B2), excluding weekends:

=NETWORKDAYS(A2, B2)

Example üí° (Excluding Holidays)

To calculate the working days, also excluding a list of holiday dates stored in the range **D2:D5**:

=NETWORKDAYS(A2, B2, D2:D5)

**Crucial Detail:** **NETWORKDAYS** is inclusive. It counts the start date and the end date if they are working days.
