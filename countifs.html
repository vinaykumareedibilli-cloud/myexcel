COUNTIFS: Count with Multiple Conditions

The **COUNTIFS** function is used to count the number of cells within a range that meet **multiple** specified criteria. This is the function you use when you need to count data based on two or more rules simultaneously (e.g., counting transactions that were completed in a specific region *and* are over a specific dollar amount).

**Key Use:** Advanced frequency counting based on complex logic.

---

Formula Syntax

=COUNTIFS(criteria_range1, criteria1, [criteria_range2, criteria2], ...)
---

Arguments (Parts of the Formula)

  • criteria_range1: The first range to apply a test against (e.g., Column A, listing Regions).
  • criteria1: The first condition (e.g., `"South"`).
  • [criteria_range2, criteria2], ...: *Optional*. Subsequent pairs of ranges and their corresponding criteria. You can add up to 127 pairs.
---

Example üí°

Count the number of sales (Column B) that were made by the "Sales Team" (Column A) **AND** were completed after the date 1/1/2025 (Column C):

=COUNTIFS(A:A, "Sales Team", C:C, ">1/1/2025")
