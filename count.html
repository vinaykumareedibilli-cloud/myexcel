COUNT: Counting Cells with Numbers

The **COUNT** function simply **counts the number of cells** in a range that contain **numerical values**. It is useful for quickly determining the size of your dataset where values are numeric.

**Important:** COUNT **ignores** cells containing text, logical values (TRUE/FALSE), or empty cells. If you need to count cells with *any* value, use **COUNTA**.

---

Formula Syntax

=COUNT(value1, [value2], ...)
---

Arguments (Parts of the Formula)

  • value1: The first argument representing the cells or range you want to count. This is required. (e.g., `A:A` or `B1:B15`).
  • [value2], ...: *Optional*. Up to 255 additional cells or ranges to include in the count.
---

Example üí°

To count how many numerical entries (like sales amounts) are present in the entire Column C:

=COUNT(C:C)
