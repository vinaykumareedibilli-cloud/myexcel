LEN: Calculating String Length

The **LEN** function returns the **number of characters** (the length) in a given text string. This count includes all characters, such as letters, numbers, punctuation, and, importantly, **spaces**.

**Key Use:** Often used with the extraction functions (**LEFT, RIGHT, MID**) to dynamically determine how many characters to extract from the end of a string.

---

Formula Syntax

=LEN(text)
---

Arguments (Parts of the Formula)

  • text: The text string or cell reference containing the text whose length you want to find (e.g., `A2`).
---

Example üí°

If A2 contains **"Product X-15"** (12 characters, including the space and hyphen), LEN returns:

=LEN(A2) → Result: **12**

Example üí° (Using with TRIM)

To find the *true* length of a cell, excluding any accidental leading or trailing spaces, you combine LEN with **TRIM**:

=LEN(TRIM(A2))
