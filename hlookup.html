HLOOKUP: Horizontal Look-up

The **HLOOKUP** function is used to **look for a value in the first row of a table** and return a corresponding value in the same column from a row you specify. The "H" stands for Horizontal.

**Key Requirement:** HLOOKUP works exactly like VLOOKUP, but searches **across the top row** instead of down the first column. This is used for data laid out horizontally (like dates or monthly totals in the top row).

---

Formula Syntax

=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
