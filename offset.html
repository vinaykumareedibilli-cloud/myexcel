OFFSET: Creating Dynamic References

The **OFFSET** function returns a reference to a range that is a specified number of **rows** and **columns** away from a starting cell (the `reference`). You can also optionally specify the `height` and `width` of the returned range.

**Key Use:** Creating dynamic charts or ranges that automatically expand or shift based on external values, often used with functions like `SUM` or `AVERAGE`.

---

Formula Syntax

=OFFSET(reference, rows, cols, [height], [width])
---

Arguments (Parts of the Formula)

  • reference: The starting cell or range from which to base the offset (e.g., `A1`).
  • rows: The number of rows to move down (positive number) or up (negative number) from the reference.
  • cols: The number of columns to move right (positive number) or left (negative number) from the reference.
  • [height]: *Optional*. The height (number of rows) of the returned reference. Default is the height of the `reference`.
  • [width]: *Optional*. The width (number of columns) of the returned reference. Default is the width of the `reference`.
---

Example üí° (Simple Offset)

Start at cell **B5**, move **2 rows down** and **1 column right**, and return the value of that single cell. (Result is the value of cell C7).

=OFFSET(B5, 2, 1)

Example üí° (Dynamic Sum Range)

Start at **A1**, move **0 rows** and **0 columns**, and create a sum range that is 1 row high and has a width determined by the value in cell **C1** (e.g., if C1=5, it sums A1:E1).

=SUM(OFFSET(A1, 0, 0, 1, C1))
