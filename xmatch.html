XMATCH: The Modern MATCH Function

The **XMATCH** function searches for a specified item in a range of cells and returns its **relative position**. It is designed to replace the older **MATCH** function by providing more flexible matching options and search directions.

**Key Use:** Like MATCH, it's often used with **INDEX** (e.g., INDEX(data, XMATCH(...))) to create advanced lookups.

---

Formula Syntax

=XMATCH(lookup_value, lookup_array, [match_mode], [search_mode])
---

Arguments (Parts of the Formula)

  • lookup_value: The value you are searching for (e.g., a specific product ID).
  • lookup_array: The single column or row that contains the value you are searching for.
  • [match_mode]: *Optional*. Specifies the type of match (0 for exact match is default, unlike the old MATCH function).
    • **0:** Exact match (default)
    • **-1:** Exact match or next smaller item
    • **1:** Exact match or next larger item
    • **2:** Wildcard match
  • [search_mode]: *Optional*. Specifies the search direction.
    • **1:** Search from first to last (default)
    • **-1:** Search from last to first (reverse)
---

Example üí° (Simple Position)

To find the position of the name in cell A2 within the list of names in column C, using the default exact match:

=XMATCH(A2, C:C)

Example üí° (Reverse Search)

Find the position of the *last* occurrence of "Closed" in the Status column (D:D):

=XMATCH("Closed", D:D, 0, -1)
