UNIQUE: Extracting a List of Distinct Values

The **UNIQUE** function is a **Dynamic Array Formula** that returns a list of **unique (distinct)** values from a range or array. It automatically removes duplicates, saving significant time previously spent on removing duplicates manually or with complex array formulas.

**Key Use:** Creating dynamic drop-down lists (Data Validation), generating master lists of categories (e.g., all regions, all product IDs), and cleaning up raw data.

---

Formula Syntax

=UNIQUE(array, [by_col], [exactly_once])

**Dynamic Array Note:** Enter the formula once in the top-left cell; the results "spill" automatically into adjacent cells.

---

Arguments (Parts of the Formula)

  • array: The range from which you want to return the unique values (e.g., the column containing product IDs, `A2:A100`).
  • [by_col]: *Optional*. A logical value: **FALSE** (default) or omitted to compare rows. **TRUE** to compare columns. Use FALSE for column data.
  • [exactly_once]: *Optional*. A logical value: **FALSE** (default) to return every unique item once. **TRUE** to return items that appear *only once* in the source data.
---

Example üí° (Simple List of Unique Regions)

To generate a list of all unique values found in the Region column (C2:C100):

=UNIQUE(C2:C100)

Example üí° (Finding Items that Appear Only Once)

To return only the names in column B that appear *just one time* in the entire list:

=UNIQUE(B2:B100, FALSE, TRUE)
