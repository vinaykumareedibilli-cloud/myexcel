TOROW: Reshaping a Range into a Single Row

The **TOROW** function is a **Dynamic Array Formula** that takes an array or range of data (which can be multiple columns/rows) and organizes all its values into a **single row**.

**Key Use:** Flattening tables, consolidating lists from several columns into one continuous row, and preparing data for functions that require a single row input.

---

Formula Syntax

=TOROW(array, [ignore_mode], [scan_by_column])

**Dynamic Array Note:** Enter the formula once; the results "spill" horizontally across cells.

---

Arguments (Parts of the Formula)

  • array: The range or array of values you want to flatten.
  • [ignore_mode]: *Optional*. Specifies which types of values to skip/ignore.
    • **0:** Include all values (default)
    • **1:** Ignore blanks
    • **2:** Ignore errors
    • **3:** Ignore both blanks and errors
  • [scan_by_column]: *Optional*. A logical value: **FALSE** (default) to read the source data **row by row** (left-to-right, then down). **TRUE** to read the source data **column by column** (top-to-bottom, then right).
---

Example üí° (Flattening a Grid - Row by Row)

If the range B2:C4 contains data: (B2=1, C2=2), (B3=3, C3=4), (B4=5, C4=6). The default scan order (row by row) returns:

=TOROW(B2:C4) → Result: **1, 2, 3, 4, 5, 6**

Example üí° (Scanning Column by Column)

Using the same data range (B2:C4), but scanning column by column (TRUE):

=TOROW(B2:C4, 0, TRUE) → Result: **1, 3, 5, 2, 4, 6**
