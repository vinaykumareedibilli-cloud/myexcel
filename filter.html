FILTER: Extracting Data Dynamically

The **FILTER** function is a **Dynamic Array Formula** that filters a range of data based on criteria you define and returns all matching rows into a spill range.

**Key Use:** Quickly creating subset reports, generating lists based on selection (like from a drop-down menu), and replacing complex `INDEX/MATCH` or `VLOOKUP` arrays.

---

Formula Syntax

=FILTER(array, include, [if_empty])

**Dynamic Array Note:** Enter the formula once in the top-left cell; the results "spill" automatically into adjacent cells.

---

Arguments (Parts of the Formula)

  • array: The range of data you want to filter (e.g., the entire data table `A2:D100`).
  • include: A logical array (TRUE/FALSE) the same height/width as the `array` that specifies which rows to include. (e.g., `C2:C100 = "West"`).
  • [if_empty]: *Optional*. The value to return if no rows meet the criteria (e.g., `"No results"`).
---

Example üí° (Single Criterion)

To return all rows from the table A2:D100 where the Region column (C2:C100) equals "East":

=FILTER(A2:D100, C2:C100 = "East", "None Found")

Example üí° (Multiple Criteria - AND Logic)

To filter for Region "East" **AND** Sales > 500. (Use the multiplication operator `*` for AND logic):

=FILTER(A2:D100, (C2:C100 = "East") * (D2:D100 > 500), "None")
