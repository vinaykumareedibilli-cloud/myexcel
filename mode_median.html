MODE & MEDIAN: Measures of Central Tendency

**Median** and **Mode** are statistical measures used alongside the **Average (Mean)** to describe the center point of a dataset.

**Median:** The middle value when data is sorted. Less affected by extreme outliers than the mean.

**Mode:** The value that occurs most frequently in the dataset.

---

1. MEDIAN Function (Middle Value)

Calculates the middle number in a set of numbers; half the numbers have values that are greater than the median, and half have values that are less.

=MEDIAN(number1, [number2], ...)
  • number1, [number2], ...: The numbers for which you want to find the median. This is typically a range of cells (e.g., `A2:A100`).

Example üí° (MEDIAN)

If the sales values in A1:A5 are {10, 5, 20, 15, 30}. (Sorted: 5, 10, **15**, 20, 30):

=MEDIAN(A1:A5) → Result: **15**
---

2. MODE Functions (Most Frequent Value)

Calculates the most frequently occurring value in a range. Excel provides several versions, but **MODE.SNGL** is the modern standard for a single result.

=MODE.SNGL(number1, [number2], ...)

**Note:** Use **MODE.MULT** if you need to return multiple modes in an array, as a Dynamic Array function.

  • number1, [number2], ...: The numbers for which you want to find the mode (e.g., a range like `B2:B50`).

Example üí° (MODE.SNGL)

If the scores in B1:B5 are {8, 10, 7, 10, 9}:

=MODE.SNGL(B1:B5) → Result: **10**
