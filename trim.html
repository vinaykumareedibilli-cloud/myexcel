TRIM: Removing Excess Spaces

The **TRIM** function removes **all extra spaces** from a text string, except for single spaces between words. This is vital for cleaning data imported from other sources that may contain leading, trailing, or multiple internal spaces.

**Key Use:** Data cleaning, ensuring consistency for lookups (VLOOKUP, XLOOKUP, etc.), and making text readable.

---

Formula Syntax

=TRIM(text)
---

Arguments (Parts of the Formula)

  • text: The text string or cell reference containing the text you want to clean (e.g., `A2`).
---

Example üí°

If A2 contains **" North Region "** (note the leading, trailing, and double spaces), TRIM cleans it up:

=TRIM(A2) → Result: **"North Region"**

**Note:** TRIM only handles standard spaces. To remove non-printing characters (like line breaks), you may need the **CLEAN** function.
